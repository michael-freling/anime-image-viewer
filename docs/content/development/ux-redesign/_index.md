# UX Redesign

The UX Redesign effort reimagines the Anime Image Viewer from the ground up, moving away from the original admin-panel layout toward a modern consumer-app experience inspired by Google Photos, Pinterest, Netflix, Spotify, Linear, and AniList.

This section groups together the design documents that describe the redesign:

- [UI Design](/development/ux-redesign/ui-design/) -- User flows, wireframes, and screen-by-screen specifications.
- [Frontend Design](/development/ux-redesign/frontend-design/) -- Component architecture, state management, and the frontend implementation plan.
